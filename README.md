Fishing Online Store
This is an online fishing store application built using HTML and CSS for the frontend, and PHP with MySQL for the backend.

Features
User registration and login: Users can create an account and securely log in.

Product ordering: Registered users can browse and order fishing products.

Blog comments: Users can add comments to blog posts related to fishing.

Admin panel: Administrators can manage orders and blog posts, including editing and deleting them.

The application provides a smooth and secure experience for both customers and administrators, allowing easy management of products, orders, and blog content.
